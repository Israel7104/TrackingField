# TrackingField

## Resumen de la aplicación
TrackingField vendría siendo una API donde se guardarían los entrenamientos ejecutados y los alimentos que el usuario consumiera.


Para este proyecto estaría recuperando un proyecto abandonado que tuve para android studio y spring boot

## Que tendria esta aplicación

La aplicación tendria dos apartados. El de nutrición, el de entrenamientos y el del usuario

### Nutrición
Aqui veriamos un buscador y una lista de productos recomendados no mas entrar. En cada producto se verian los macros, un boton para colocar si nos hemos comido ese alimento, etc...

### Entrenamientos
Aqui veriamos ejercicios. Cual es nuestro mr teorico en cada uno y nos dejaria crear rutinas. En las rutinas te dejaria poner cuantas series y repeticiones quieres hacer con el porcentaje del mr que decidas colocar, etc...

### Usuario
Aqui veriamos cosas como cuanto pesamos exactamente, cuantas calorias hemos consumido de media cada semana, cuantas sesiones hemos tenido de entrenamiento, etc...
