- Agile es el concepto sobre que que hay que tener un plan de hacer cosas pequeñas para hacer algo grande.
- Kanban es el metodo para poder gestionar el trabajo rapidamente de forma visual viendo que esta completado, que esta por completar y que se tendra que empezar.
- Scrum es un metodo para trabajar de forma ordenada en equipo planificando mini entregas con fechas previstas.


- Las diferencias entre kanban y scrum son que kanban se centra en que el trabajo continue siempre buscando la flexibilidad sin fechas definidas mientras que scrum busca tenerlo todo definido de forma exacta.
